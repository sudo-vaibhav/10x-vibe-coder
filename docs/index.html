// The Problem

AI coding tools like Cursor, Conductor, and VS Code + Copilot are optimized for conversational input. Voice-to-text is 3-5x faster than typing for giving instructions to AI agents.

Yet developers habitually type out of muscle memory, missing out on massive productivity gains.

// The Solution

NUDGE

Gentle reminders

When you exceed a keystroke threshold, Vibe10X displays a non-intrusive alert reminding you to use voice input.

HABIT

Build new patterns

Over time, you'll naturally reach for voice input first. The nudges help rewire muscle memory for the AI coding era.

// Philosophy

" The best tool for AI coding is your voice. Vibe10X helps you remember that. "

Typing 100+ WPM feels fast, but speaking at 150 WPM while the AI does the heavy lifting is 10X faster.

Vibe10X isn't about restriction. It's about building awareness of when voice input would serve you better. The nudge is gentle, the habit change is powerful.

// Quick Start

terminal
# Install Hammerspoon first
$ brew install hammerspoon
# Clone and install
$ git clone https://github.com/vaatun/vibe10x.git
$ cd vibe10x
$ bun install
$ bun link
# Run interactive setup
$ vibe10x
# Or use a preset
$ vibe10x --preset zen
